Class: Visitor

Visitor(loggeropt)

Clase que implementa un visitante para el lenguaje Bies. Extiende la funcionalidad de `biesLanguageVisitor` para recorrer un árbol de sintaxis y generar las instrucciones correspondientes que se ejecutarán en la máquina virtual (VM).
