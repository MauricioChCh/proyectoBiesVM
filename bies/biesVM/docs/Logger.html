Class: Logger

Logger(isVerboseopt, isDebugopt, isOutfileopt, isErrfileopt)

Clase `Logger` que proporciona funcionalidades para registrar mensajes en la consola. Permite activar diferentes niveles de detalle en los logs a través de las propiedades `isVerbose`, `isDebug`, `isOutfile` e `isErrfile`.

Constructor

new Logger(isVerboseopt, isDebugopt, isOutfileopt, isErrfileopt)

Crea una nueva instancia de la clase `Logger`. Esta función solo se ejecuta una vez para crear la instancia.
Parameters:
Name Type Attributes Default Description
isVerbose boolean <optional>
 false Indica si se deben mostrar mensajes de registro detallados.
isDebug boolean <optional>
 false Indica si se deben mostrar mensajes de depuración.
isOutfile boolean <optional>
 false Indica si se deben registrar mensajes en un archivo de salida.
isErrfile boolean <optional>
 false Indica si se deben registrar mensajes de error en un archivo.
