Class: Logger

Logger(isVerboseopt, isDebugopt)

Clase `Logger` que proporciona funcionalidades para registrar mensajes en la consola. Permite activar diferentes niveles de detalle en los logs a través de las propiedades `isVerbose` y `isDebug`.

new Logger(isVerboseopt, isDebugopt)

isVerbose boolean <optional>
 false Indica si se deben mostrar mensajes de registro detallados.
isDebug boolean <optional>
 false Indica si se deben mostrar mensajes de depuración.
